# Basic Image Manipulation with Pillow

This project implements fundamental **Image Manipulation** tasks using the **Pillow (PIL)** library. It focuses on the essential skills needed to open, edit, and save images using Python’s most popular imaging library.

## 🚀 Overview
The goal of this project is to demonstrate the core functionality of Pillow. By treating images as objects, the application performs straightforward operations such as resizing, rotating, and format conversion, providing a simple entry point into digital image handling.

## Key Features
* **File Operations:** Opening various image formats (JPEG, PNG, BMP) and saving them with custom quality settings.
* **Basic Transformations:** Simple methods for resizing, rotating by degrees, and flipping images horizontally or vertically.
* **Color Mode Conversion:** Converting images between **RGB**, **Grayscale (L)**, and **Binary (1)** modes.
* **Region Copy & Paste:** Cutting specific areas from an image and pasting them into another or back onto the original.
* **Image Synthesis:** Creating new blank images from scratch with specific dimensions and background colors.
